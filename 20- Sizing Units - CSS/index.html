Sizing Units

What are sizing units?

Sizing units are used to specify the height and width of elements. Eg. px, vh, vw, vmin, vmax etc.

Every sizing unit has different specifications. Some of them are defined below.

Px
Px means pixel values.
VH
VH means percentage of view port height.
VW
VW means percentage of view port width.
VMin
VMin is used with the minimum dimentions of the device. For mobiles Vmin is width and for computer devices it's height.
VMax
VMax is used with max dimentions of the device. For mobiles Vmax is height and width for computer devices.

Types of CSS

What are the types of CSS or different ways to add CSS?

There are 3 types of CSS or 3 different ways to add CSS to your page.

  1. Inline CSS
  2. Internal CSS
  3. External CSS

1. Inline CSS

                    
    <h1> style="background-color: rgb(5, 97, 100); color: wheat; padding: 8px;">Types of CSS</h1>

2. Internal CSS

When we use "style" tag in the head section of the same file where we need to style the elements, it's called Internal CSS.

                    
    <!DOCTYPE html>
    <html lang="en">
    <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Internal CSS</title>
            <style>
                    *{
                            margin: 0px;
                    }
                    h2{
                            background-color: black;
                            color: white;
                    }
            </style>
    </head>
    <body>
            <h2>This is an example of Internal CSS</h2>
    </body>
    </html>

3. External CSS

When we define CSS in a seprated file and link this file in the document by using <link rel="stylesheet" href="filename.css">

HTML

                    
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>External CSS</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <h1>External CSs</h1>
        <p>This is an example of external CSS.</p>
    </body>
    </html>

CSS

                    
    style.css
    h1{
        color: blue;
    }

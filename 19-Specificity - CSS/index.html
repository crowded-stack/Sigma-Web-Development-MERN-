Specificity

What is specificity?

Specificity means when an element has multiple references for styling in css and there is need to decide which will be applied. An algorithm takes place to resolve this conflict called "Cascade Algorithm".

Stages of Cascade Algorithm

Cascade algorithm defines 4 stages to resolve the conflict.

Position and Order of Appearence
Let's suppose, an element has 4 references of same type of selector. Coming at the end in the sequence will be applied.
Specificity
It decides according to the priority of css selectors. Which has high priority, it will be applied.
Important --> Inline --> Id Selector --> Class/Attribute/Pseudo Class Selectors --> Element/Tag Selector --> Universal Selector
Origin
When we do not define any style for element then the styles are provided by the browser. It's called user agent style sheet.
Important
When a selector is defined with a keyword "important" it will has high priority and will be applied rather than others

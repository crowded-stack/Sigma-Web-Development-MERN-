Colors

5 ways to define colors in css.

Color Keyword
                h1{
                    color: blue;
                }
Hex Color Code
                h1{
                    color: #6f899a;
                }
RGB
                
Simple RGB

                h1{
                    color: rgb(233, 122, 34);
                }

                
RGB with Alpha (opacity range: 0-1, default is 1).

                h1{
                    color: rgba(233, 122, 34, 1);
                }
HSL
                
HSL also 2 variants, with alpha and without alpha.

                h1{
                    color: hsl(233, 100%, 100%);
                }

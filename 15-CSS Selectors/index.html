CSS Selectors

What are CSS selectors?

CSS selectors are used to select elements in different ways to style them. There are various number of selectors. Few of them are written below.

Universal Selector
This is used to select the whole document. 
                            
            * {
                background-color: black;
                margin: 0;
            }
Element Selector
This is used to select element by using tag name 
                            
            h1 {
                background-color: black;
                color: white;
            }
Id Selector
This is used to select element by using id defined within tag. 
                            
            #id_of_element {
                background-color: black;
                color: white;
            }
Class Selector
This is used to select element by using class defined within the tag. 
                            
            .class_name {
                background-color: black;
                color: white;
            }
Direct-children Selector
This is used to select elements those are directly children of parent. 
                            
            dl > p {
                color: blueviolet;
            }
Children Selector
This is used to select elements those are directly or indirectly children of parent. 
                            
            dl code {
                color: chocolate;
            }
